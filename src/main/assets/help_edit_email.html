Permissions description

Edit E-mail

Here you can edit an E-mail record.

The description field is optional. If it is filled in, a Smart Poster record is created, if not a URI record is created.

The format of the URI that is used is sms:<address>?subject=<subject>&body=<body text>

Depending on whether you are editing or creating a record, there will be different buttons visible:

  • Next - start writing
  • Save - finish editing
