Permissions description

Edit Bluetooth record

Here you can edit a bluetooth record.

The MAC address you can usually find on a sticker on the device, or when scanning it from your PC. It should have the format xx:xx:xx:xx:xx:xx

Optionally, you can enter a description for the device that this MAC address belongs to.

Click the 'From this phone' button to take the MAC address and bluetooth name from this phone.

Depending on whether you are editing or creating a record, there will be different buttons visible:

  • Next - start writing
  • Save - finish editing
