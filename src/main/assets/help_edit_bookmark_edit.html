Permissions description

Edit bookmark

Here you can edit a bookmark

The title field is optional. If it is filled in, a Smart Poster record is created, if not a URI record is created.

The prefix http://www. will be automatically prepended to your string.

Depending on whether you are editing or creating a record, there will be different buttons visible:

  • Next - start writing
  • Save - finish editing

You can use special values in your fields that will be replaced during writing. The following values are supported:

  • %ctr1% - The first counter. The value starts at 0 and is increased every time this counter is used. To reset the counter, go to the preferences screen.
  • %ctr2% - The second counter.
  • %ctr3% - The third counter.
  • %tagid% - The ID of the NFC item being written to, in hexadecimal format. The size depends on the item type.
