Tools menu

The tools menu offers some meta-operations on NFC items:

  • Erase: Quickly erase the contents of an NFC item by writing an empty message.
  • Full Erase: Quickly erase the full contents of an NFC item by writing default data.
  • Protect: Protect an NFC item so it cannot be overwritten anymore. Note that this is not supported by all NFC items so it may fail for yours.
    IMPORTANT: This is a one-time operation that cannot be undone
  • Clean: Fully erase the contents of an NFC item by fully overwriting its content.
