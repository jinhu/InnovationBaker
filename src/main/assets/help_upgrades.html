Upgrades

In this screen you can purchase upgrades for TagWriter.

TagWriter features are split into 3 categories:

  • Lite: Write and share various content to NFC items
  • Standard: Automatic backups, write more types of content, format/protect items
  • Pro: Use variables in URLs, repeating write, protect after write

The drop-down has a list of available upgrades. Select one to display a description of the upgrade (which features will be unlocked).

If you decide to upgrade, click the 'Upgrade' button which will open the Android Market and allow you to purchase the feature set.

After purchasing an upgrade, restarting TagWriter may be required to enable all new functionality.
